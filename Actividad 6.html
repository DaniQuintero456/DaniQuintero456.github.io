Siglas

OSI (Open Systems Interconnection)

Modelo conceptual de 7 capas que estructura cómo los sistemas se comunican en redes.

IP (Internet Protocol)

Protocolo para direccionar y enrutar paquetes de datos en redes.

TCP (Transmission Control Protocol)

Protocolo fiable orientado a conexión para la transmisión de datos.

UDP (User Datagram Protocol)

Protocolo más rápido, pero menos fiable, orientado a datagramas sin conexión.

SSH (Secure Shell)

Protocolo de red que permite conectarse a un servidor a través de una red no segura

FTP (File Transfer Protocol)

Es un protocolo de red que permite transferir archivos entre computadoras conectadas a una red, como Internet.

SMTP(Simple Mail Transfer Protocol)

Es un protocolo de comunicación que se utiliza para enviar y recibir correos electrónicos por internet.

POP (Post Office Protocol)

Protocolo para descargar correos desde un servidor

DHCP (Dynamic Host Configuraton Protocol)

Protocolo de configuración dinámica de host. Es un protocolo de red que asigna direcciones IP a los dispositivos de una red

DNS (Domain Name System)

Sistema que asigna nombres de dominio a dircciones IP numericas

HTTP (Hypertext Transfer Protocol)

Es un protocolo que permite la comunicación entre navegadores y servidores web.

ARP (Address Resolution Protocol)

Protocolo que vincula una direccion IP a una direccion IP a una direccion MAC

Media Access Control (MAD)

Codigo Unico que identifica a un dispositivo en una red

LAN (Local Area Network)

Se tratra de una red de computadoras que permite la comunicacion entre dispositivos cercanos, como los del hogar o una oficina
